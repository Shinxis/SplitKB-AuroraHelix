# SplitKB Aurora Helix
An ergonomic split keyboard with ortholinear grid layout, using Aurora Helix from [splitkb.com](https://splitkb.com)<br>
The design files for the case include:<br>
Top plate, Bottom plate, Controller cover, Mold for making an epoxy pour of the bottom plate.<br>

FreeCAD model
 exported .stl files, and Bambu Studio project files (excluding bottom plate).
